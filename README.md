# pangyoslang

판교 사투리를 일상어로 번역해주는 iOS 앱입니다.

`pangyoslang`은 IT 업계에서 자주 쓰이는 영어 기반 업무 표현, 줄임말, 업계 용어를 일상어로 풀어주는 앱입니다.  
판교 지역의 IT 직장인들이 업무 과정에서 사용하는 표현들이 일상 대화 속에서도 자연스럽게 섞이는 현상을 재미있게 바라보고, 이를 ‘판교’와 ‘사투리’라는 단어를 합쳐 **판교 사투리**라는 콘셉트로 풀어냈습니다.

이 프로젝트는 Apple Developer Academy의 세 번째 Challenge 과정에서 진행하는 팀 프로젝트입니다.  
Apple의 공식 기술인 **Speech**를 활용해 사용자의 발화를 텍스트로 변환하고, 문장 안에 포함된 판교어를 감지한 뒤 일상어로 번역하는 경험을 만드는 것이 핵심 목표입니다.

---

## 프로젝트 개요

### 앱 이름

**판교 사투리**

### 프로젝트 이름

**pangyoslang**

### 핵심 아이디어

IT 업계에서 자주 사용되는 업무 용어는 때로 일반적인 일상 대화에서는 낯설게 느껴질 수 있습니다.  
예를 들어 “얼라인”, “아삽”, “싱크 맞추기”, “테스크 플로우” 같은 표현은 특정 업무 환경에서는 자연스럽지만, 익숙하지 않은 사람에게는 바로 이해하기 어려울 수 있습니다.

`pangyoslang`은 이런 판교어를 단순히 해석하는 데 그치지 않고, 사용자가 새로운 표현을 접하며 웃음과 재미를 느낄 수 있도록 설계된 앱입니다.

---

## 주요 기능

### 1. 판교어 문장 번역

사용자가 판교어가 섞인 문장을 말하면, 앱이 음성을 텍스트로 변환하고 문장 안에 포함된 판교어를 감지합니다.  
감지된 판교어는 일상어로 대치되어 더 이해하기 쉬운 문장으로 보여집니다.

기본 흐름은 다음과 같습니다.

1. 사용자가 판교어가 섞인 문장을 발화합니다.
2. Speech를 이용해 음성을 텍스트로 변환합니다.
3. 문장의 구조를 나누고 단어 단위로 분석합니다.
4. 판교어를 감지합니다.
5. 감지된 판교어를 일상어로 대치합니다.
6. 번역된 문장을 사용자에게 보여줍니다.

---

### 2. 판교어 사전

사용자가 궁금한 판교어를 직접 검색할 수 있는 사전 기능입니다.

사전에서는 다음 정보를 제공합니다.

- 판교어 표현
- 일상어 뜻
- 사용 예문
- 관련 태그 또는 카테고리

일반적인 언어 사전처럼 특정 단어의 뜻을 찾아보는 경험을 제공하되, 판교어 특유의 밈적이고 가벼운 분위기를 함께 담는 것을 목표로 합니다.

---

### 3. 오늘의 판교어

메인 화면에서는 매일 랜덤으로 하나의 판교어를 보여줍니다.

사용자는 앱을 실행했을 때 새로운 판교어와 그 뜻, 예문을 확인할 수 있습니다.  
이를 통해 평소에 사용하지 않았던 표현이나 최근 자주 쓰이는 업무 용어를 가볍게 접할 수 있습니다.

이 기능은 단순한 정보 전달을 넘어, 사용자가 앱에 들어왔을 때 작은 웃음과 재미를 느끼도록 하기 위한 장치입니다.

---

## 태스크 플로우

현재 앱의 기본 태스크 플로우는 크게 세 가지 흐름으로 구성됩니다.

### 1. 랜덤 판교어 알림

- 메인 화면에서 오늘의 판교어를 랜덤으로 제공합니다.
- 판교어, 뜻, 예문 데이터를 기반으로 보여줍니다.

### 2. 판교어 번역

- 사용자가 마이크 권한을 허용합니다.
- 판교어가 섞인 문장을 발화하거나 입력합니다.
- Speech를 통해 음성 데이터를 텍스트로 변환합니다.
- Tokenization을 통해 문장을 단어 단위로 분석합니다.
- 판교어 사전 데이터와 비교해 판교어를 감지합니다.
- 감지된 판교어를 일상어로 치환합니다.
- 변환된 문장을 결과 화면에 표시합니다.
- 필요할 경우 TTS 버튼을 통해 변환된 문장을 음성으로 들을 수 있습니다.

### 3. 판교어 사전

- 사용자가 사전 탭에 진입합니다.
- 판교어 카테고리 또는 검색 기능을 통해 단어를 찾습니다.
- 검색 결과에서 원하는 단어를 선택합니다.
- 해당 단어의 뜻과 예문을 확인합니다.
- 학습 완료 흐름으로 연결될 수 있습니다.

## 사용 기술

### Apple Frameworks

- **SwiftUI**  
  앱의 전체 화면과 인터랙션을 구성합니다.

- **Speech**  
  사용자의 음성을 텍스트로 변환하는 데 사용합니다.

- **Natural Language**  
  문장을 단어 단위로 나누고, 판교어 감지 및 태깅 흐름에 활용할 수 있습니다.

- **AVFoundation**  
  변환된 문장을 음성으로 읽어주는 TTS 기능에 활용할 수 있습니다.

## 아키텍처 방향

이 프로젝트는 SwiftUI에 맞는 구조로 설계합니다.

화면 코드는 최대한 UI 표현에 집중하고, 판교어 데이터 처리와 번역 로직은 별도의 역할로 분리하는 것을 목표로 합니다.

예상 구조는 다음과 같습니다.

```text
pangyoslang
├── App
├── Models
├── Views
├── ViewModels
├── Services
├── Resources
└── Data
```

### App

앱의 진입점과 전역 설정을 관리합니다.

### Models

판교어 데이터 구조를 정의합니다.

예시:

- 판교어
- 일상어 뜻
- 예문
- 태그
- 카테고리

### Views

SwiftUI 화면을 구성합니다.

예시:

- 메인 화면
- 번역 화면
- 사전 화면
- 검색 결과 화면
- 판교어 상세 화면

### ViewModels

화면에 필요한 상태와 사용자 액션을 관리합니다.

예시:

- 오늘의 판교어 불러오기
- 검색어 입력 처리
- 번역 결과 상태 관리
- 로딩 및 에러 상태 관리

### Services

실제 기능 로직을 담당합니다.

예시:

- 음성 인식 처리
- 판교어 감지
- 판교어 번역
- TTS 처리
- 사전 데이터 검색

### Resources / Data

앱에서 사용하는 판교어 사전 데이터를 관리합니다.

초기 버전에서는 JSON 기반의 로컬 데이터를 사용할 수 있으며, 이후 필요에 따라 외부 데이터 소스나 서버 기반 구조로 확장할 수 있습니다.

---

## 데이터 예시

```json
{
  "term": "얼라인",
  "meaning": "의견이나 방향을 맞추다",
  "example": "이번 기획 방향 먼저 얼라인하고 진행하자.",
  "translatedExample": "이번 기획 방향 먼저 맞추고 진행하자.",
  "tags": ["업무", "회의", "소통"]
}
```

## 프로젝트 목표

`pangyoslang`의 목표는 판교어를 모르는 사람도 쉽게 이해할 수 있도록 돕는 것입니다.  
동시에 IT 업계 특유의 언어 문화를 무겁지 않게 풀어내고, 사용자가 새로운 표현을 배우는 과정에서 재미를 느끼게 하는 것을 지향합니다.

단순한 번역 앱이 아니라, 판교어라는 밈과 업무 언어 문화를 사전, 번역, 랜덤 학습 경험으로 연결하는 iOS 앱을 만드는 것이 이 프로젝트의 핵심입니다.
